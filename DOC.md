## Things to be considered while developing API's for Garuda

1. Every line of function call in C should reflect Garuda's code.
   This makes it easy for developing instruction pattern.

2. Make sure to include the anticipated Garuda LOC in comments while
   calling the function. i.e., specify the instruction pattern.

3. Consider going through few of the source codes to understand the
   instrcution pattern followed by Garuda.

4. Access (SourceCodes)[https://github.com/rohitpidishetty/garuda-source-codes]
